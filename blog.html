## Discuss the scope of var, let, and const

# var is function scoped. A variable decleared with var keyword can be access from anywhere inside a function.


# let is block scoped. A variable decleared with let keyword can be access only from the block from which it is decleared. A variable decleared with let keyword can be reassigned with new values.


# const is block scoped. A variable decleared with const keyword can be access only from the block from which it is decleared. It is different from let because a variable decleared with const keyword can not be reassigned with new values.

## What do you mean by REST API?

# REST means "Representational State Transfer". Nowadays, everyone is using REST API for exchanging information securely over the internet between two computers. A REST API basically performs FOUR operations on any data, which "Create", "Read", "Update", "Delete". In short this operations are called CRUD operations . REST API uses mainly FIVE HTTP Methods. The HTTP Methods are:

  • GET: The HTTP GET Method is used to "Read" a data from API.
  • POST: The HTTP POST Method is used to "Create" a data using API.
  • PUT|PATCH: The HTTP PUT|PATCH Method is used to "Update" a data using API.
  • DELETE: The HTTP DELETE Method is used to "Delete" a data using API.
